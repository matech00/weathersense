# WeatherSense AI 🌤️

> An AI-powered weather application that provides real-time weather updates, location-based forecasts, and personalized weather insights.

WeatherSense AI is a modern weather application that provides real-time weather information based on the user's location. The application uses geolocation technology and weather APIs to display current weather conditions, temperature, humidity, wind speed, and local time. It also includes an AI-powered weather assistant that analyzes weather data and provides useful recommendations for users.

[View Live Demo](https://weathersense.vercel.app)

## ✨ Features

- Automatic Location Detection\*\* — Get weather instantly based on your location
- Real-time Weather Updates\*\* — Current temperature, humidity, wind speed, and conditions
- City Search\*\* — Search for any city worldwide
- 5-Day Forecast\*\* — Plan ahead with weather predictions
- Hourly Forecast\*\* — Detailed weather breakdown by hour
- Air Quality Index\*\* — Monitor air quality with pollutant data
- AI Weather Assistant\*\* — Personalized recommendations for clothing, activities, and hydration
- Weather Alerts\*\* — Automatic warnings for extreme weather conditions
- Dynamic Radar\*\* — Live weather radar imagery
- Dark/Light Mode\*\* — Toggle between themes
- Favorites\*\* — Save cities for quick access
- Search History\*\* — Keep track of recent searches
- Share Weather\*\* — Share conditions with others
- Fully Responsive\*\* — Works on desktop, tablet, and mobile

## 🛠️ Technologies Used

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- Vanilla JavaScript (ES6+)
- OpenWeatherMap API
- Geolocation API
- Font Awesome Icons
